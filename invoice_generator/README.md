# MN STEEL DOOR — Invoice Generator

مولّد فواتير احترافي لأعمال تصدير الأبواب الصلب. يعمل محلياً على جهازك (localhost) فقط — غير منشور على الإنترنت.

---

## 🚀 التشغيل (Mac)

افتح **Terminal** ونفّذ الأوامر بالترتيب:

```bash
# 1) ادخل لمجلد البرنامج
cd ~/Desktop/door_ops_app/invoice_generator      # أو المسار اللي حطّيت فيه المجلد

# 2) مكتبة توليد PDF تحتاج Pango (مرة واحدة فقط)
brew install pango

# 3) أنشئ بيئة بايثون وثبّت المتطلبات
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 4) شغّل البرنامج
python app.py
```

ثم افتح المتصفح على:  **http://localhost:8000**

> في كل مرة لاحقاً: `cd` للمجلد ثم `source .venv/bin/activate` ثم `python app.py`.

- لتغييرها لاحقاً: سجّل خروج ثم ادخل، أو احذف ملف القاعدة (يعيد ضبط كل شيء).

## 📁 أين تُحفظ البيانات؟

كل شيء داخل مجلد واحد: **`~/Desktop/door_ops/`**
- `invoice.db` — قاعدة البيانات (العملاء، الشركات، الفواتير، الأسعار).
- `invoices/[اسم العميل]/` — كل فاتورة محفوظة PDF (نسخة Proforma + Commercial).
- `backups/` — نسخة احتياطية تلقائية للقاعدة قبل أي حذف.

**نسخة احتياطية يدوية:** انسخ ملف `~/Desktop/door_ops/invoice.db` لأي مكان آمن.

## ✅ الميزات

- اختيار العميل + الشركة المصدّرة + بنود الأبواب → فاتورة احترافية جاهزة للطباعة/PDF.
- نسختان لكل فاتورة (Proforma / Commercial) بضغطة.
- **حاسبة الكونتينر** الذكية + تنبيه النقص.
- **ذاكرة الأسعار** (تتذكر آخر سعر لكل موديل ولكل عميل).
- **حارس الربح** (هامش بالألوان أخضر/أصفر/أحمر — سرّي).
- **سطر العمولة** السرّي (متدرّج حسب العرض / 3$ للمتر² للعراق).
- أرشفة تلقائية منظّمة + سجل بحث + تكرار فاتورة كقالب.
- تذكير **NAFEZA** للعملاء المصريين + حقل ACID إلزامي.
- تعدد العملات (USD أساسي + معادل لحظي عبر frankfurter.app).

## ✍️ ثلاثة أشياء تملؤها بنفسك

من صفحة **الإعدادات** داخل البرنامج:

1. **الشركتان المصدّرتان 2 و 3** — بياناتهما فارغة جاهزة للتعبئة (الأولى Sairogullari مكتملة).
2. **أسعار الموديلات** — السعر الافتراضي لكل موديل (يظهر تلقائياً عند الفوترة).
3. **أسعار التكلفة (سرّية)** — لكل موديل، لتفعيل حارس الربح (لا تظهر أبداً في فاتورة العميل).

> العملاء (92 عميل) محمّلون مسبقاً من جهات اتصالك — تقدر تعدّل/تحذف أي واحد، وتضيف جدد من شاشة الفاتورة.
