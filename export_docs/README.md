# AKVA Export Docs (Pro) — MN STEEL DOOR

نظام توثيق تصدير احترافي. بكبسة زر يولّد عدّة وثائق تصدير PDF **ويرتّبها تلقائياً** بمسارات منظّمة داخل
`~/Desktop/TLT_DOOR_ORGANIZED/`. توليد PDF مباشر (WeasyPrint) — **بدون Print Center، بدون كراش**.

نسخة أقوى وأوضح من الأداة السابقة: ٥ شركات مصدّرة، ٥ أنواع وثائق، جدول حاويات للباكنج ليست،
معاينة حيّة لمسار الحفظ، تنبيه NAFEZA، سجل وتكرار.

---

## 🚀 التشغيل (Mac)

```bash
cd ~/Desktop/door_ops_app/export_docs
brew install pango
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

ثم افتح: **http://localhost:8000** — كلمة السر الأولى **`MN2026`** (يطلب تغييرها).

> macOS يحجز البورت 5000 لـ AirPlay، فالافتراضي 8000. لتغيير البورت: `python app.py 8050`.

## 📂 المخرجات المنظّمة

كل وثيقة تُحفظ **Excel (.xlsx)** تلقائياً (+ نسخة PDF للطباعة/التوقيع) في:

```
~/Desktop/TLT_DOOR_ORGANIZED/[اسم الزبون]/[YYYY-MM]/[نوع الوثيقة]/
        YYYY-MM-DD_نوع_الوثيقة_الزبون_AKVA2026-XXXX.xlsx   ← الصيغة الأساسية
        YYYY-MM-DD_نوع_الوثيقة_الزبون_AKVA2026-XXXX.pdf    ← نسخة إضافية
```

الملف Excel منسّق ومرتّب جاهز للتعديل، وكل وثيقة في شيت مستقل.

أنواع الوثائق المتاحة:
- **Commercial Invoice** — فاتورة تجارية
- **Proforma Invoice** — بروفورما
- **Packing List** — قائمة تعبئة (مع الحاويات والأوزان)
- **Shipping Instructions** — تعليمات شحن
- **Certificate of Origin** — شهادة منشأ

بيانات البرنامج (قاعدة البيانات + النسخ الاحتياطية) في `~/Desktop/door_ops/akva.db`.

## ✅ المميزات

- شاشة واحدة: شركة مصدّرة + زبون + أبواب + حاويات + اختيار الوثائق → **توليد بكبسة**.
- ٥ شركات مصدّرة (SAIROGULLARI جاهزة، والباقي تملؤها من الإعدادات).
- قائمة زبائنك جاهزة + إضافة زبون جديد فوراً.
- جدول أبواب مرن (نوع/عرض/ارتفاع/سماكة عارضة/عدد/سعر) + جدول حاويات للباكنج ليست.
- معاينة حيّة لمسار الحفظ قبل التوليد.
- تنبيه **NAFEZA** للزبائن المصريين + ACID إلزامي.
- تعدد عملات (USD أساسي + معادل لحظي frankfurter.app).
- سجل بحث + **تكرار** أي شحنة كقالب + حذف بتأكيد ونسخة احتياطية.
- توليد PDF مباشر وموثوق — لا يعتمد على Print Center.

## ✅ الشركات والزبائن — معبّأة مسبقاً

الشركات الخمسة معبّأة بالكامل (الاسم القانوني، العنوان، الهاتف، الإيميل، الرقم الضريبي VKN، والبنك):
**SAIROGULLARI · TLT · RANGE · NEWA GROUP · MFY**. والزبائن محمّلون بأسماء عربية واضحة
(مع ACID الثابت لـ "سعد زكي — Modern Style" يُعبّأ تلقائياً عند اختياره).

## ✍️ ما يبقى لتكمّله (اختياري، من الإعدادات)

1. **بيانات بنكية ناقصة**: IBAN لـ RANGE، وبنك/هاتف TLT و MFY (إن وُجدت).
2. **أسعار الموديلات** الافتراضية.
3. **شروط الدفع** الافتراضية (الحالية: CASH AGAINST DOCUMENTS 100%).

> أي تعديل على شركة أو زبون من صفحة الإعدادات يبقى محفوظاً ولا يُستبدل عند التحديث.
